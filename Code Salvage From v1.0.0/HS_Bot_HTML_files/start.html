Administrator Commands /poweroff - Temporarily deactivates the system. /restart - Restarts the system Stopwatch Commands /newstopwatch - Creates a new stopwatch. /checkstopwatch - Gets the status of a stopwatch. /resetstopwatch - Resets a stopwatch. /deletestopwatch - Deletes a stopwatch.
